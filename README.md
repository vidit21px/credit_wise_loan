# Credit wise loan system
This is a ml project which classifies whether a loan application should be approved or not

# Dataset
The dataset was downloaded from kaggle 
It had 19 features and 1 target feature
It had 1000 entries
Alot of data cleaning and data transformation was performed to make model more accurate

# EDA
Multiple charts and plot were made to analyze and vizualize the data
Heatmap and corelation matrix was created to check important features

# Models
Logistic Regression gave accurcay of 86.5%
Navie Bayes gave a accurcay of 86.5%
Knn gave a accuracy of 76.5%
